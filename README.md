# RDB-Net: Regularized Depthwise Bottleneck Network for Colonoscopy Polyp Segmentation
Official Project Repository for the manuscript submitted to *Biomedical Signal Processing and Control*

## Currently Available Repository Contents
### 1. Public Benchmark Datasets
All experiments in this work are evaluated on five publicly available colonoscopy polyp segmentation datasets:
1. Kvasir-SEG
2. CVC-ClinicDB
3. CVC-ColonDB
4. ETIS-LaribPolypDB
5. CVC-300

Official dataset download sources, standard data splitting rules, and brief statistical descriptions are provided in this repository to ensure experimental transparency.

### 2. Model Output Prediction Maps & Quantitative Results
We have released the final segmentation prediction maps generated by our proposed RDB-Net model on all test sets of the above five public datasets.
Standard segmentation evaluation metrics including Dice Similarity Coefficient (DSC), Intersection over Union (IoU), Precision, Recall and Accuracy are also documented for reference and result verification.

### 3. Pre-trained Model File
The saved weight file of our proposed RDB-Net model is included in this repository.
